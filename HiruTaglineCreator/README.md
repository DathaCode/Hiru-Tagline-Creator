# Hiru News Tagline Creator

Professional broadcast tagline graphics generator for TV news production.

## Version
1.1

## Features
- Dual template support (MAIN TAG / SUB TAG)
- Automatic Sinhala Unicode conversion
- Live preview with text overlay
- Session-based workflow
- Selective PNG generation
- Manual text adjustments
- Text validation and warnings

## System Requirements
- Windows 10 or later
- Python 3.11+ (for development)
- Required fonts: FM GANGANEE, FM SANDHYANEE
- 4GB RAM minimum
- 500MB free disk space

## Installation
See USER_GUIDE.md for detailed instructions.

## Development
This project uses Python + Tkinter for the GUI and Pillow for image generation.

### Project Structure
See project tree in docs/

### Running from Source
```bash
pip install -r requirements.txt
python main.py
```

### Building Executable
```bash
python build_exe.py
```
