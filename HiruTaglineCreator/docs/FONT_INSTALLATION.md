# Font Installation Guide

## Required Fonts
- FM GANGANEE (for Topic Bed)
- FM SANDHYANEE (for TAG and White Beds)

## Installation Steps

### Windows 10/11
1. Download font files (.ttf)
2. Right-click on font file
3. Select "Install" or "Install for all users"
4. Restart Hiru News Tagline Creator

### Font Location
Fonts must be installed in:
`C:\Windows\Fonts\`

## Verification
Open Settings → Fonts tab to verify installation status.

## Troubleshooting
If fonts don't appear:
- Ensure .ttf files are valid
- Try restarting Windows
- Check Windows Fonts folder
